Notenrechner für Lehrkräfte — was die App kann

Drei Bereiche, ein Ziel: die Notenverwaltung für Ihren Schulalltag erleichtern.

Der Notenrechner ist eine kostenlose App für Lehrkräfte, mit der Sie Notenschlüssel berechnen, Klassen verwalten und Zeugnisnoten generieren — offline, ohne Tracking und ohne Werbung.

🖩

Notenrechner

Geben Sie Punkte ein und erhalten Sie sofort die Note — mit IHK, Notenpunkten oder einem per Generator erstellten Notenschlüssel.

🏫

Klassenverwaltung

Erstellen Sie Klassen, fügen Sie Schüler hinzu, verwalten Sie Klausuren, Teilnoten, mündliche Noten und generieren Sie Zeugnisnoten — alles automatisch berechnet.

📊

Notenübersicht

Sehen Sie alle Notenschlüssel, Grenzen und Notentabellen auf einen Blick. Perfekt für die schnelle Einsicht vor der Klausur.

Flexible Notenschlüssel

Wählen Sie zwischen IHK-Notenschlüssel, Notenpunkten (15–0) und linearen Schlüsseln. Oder erstellen Sie mit dem Generator Ihren eigenen: Bestehensgrenze per Schieberegler einstellen, optional einen Sockel setzen — die App berechnet alle 16 Notenstufen automatisch. Drei Rundungsmodi sorgen für maximale Flexibilität.

IHK Notenpunkte Generator mit Knick 3 Rundungsmodi

Zeugnisnoten auf Knopfdruck

Gewichtete Berechnung aus Klausuren, Teilnoten und mündlicher Mitarbeit. Stellen Sie individuelle Gewichtungen pro Klausur ein und überschreiben Sie bei Bedarf einzelne Noten manuell. Export als PDF inklusive.

Alles rund um Notenschlüssel & Notenverwaltung

Antworten auf die häufigsten Fragen von Lehrkräften an Gymnasium, Realschule, Hauptschule und Berufsschule.

Ein Notenschlüssel definiert, wie viele Punkte für welche Note nötig sind. Er legt die Prozentgrenzen fest, ab denen eine bestimmte Note vergeben wird — z. B. ab 50 % eine 4, ab 81 % eine 2. Lehrkräfte an Gymnasien, Realschulen, Hauptschulen und Berufsschulen verwenden unterschiedliche Schlüssel je nach Schulform und Bundesland.
Beim IHK-Notenschlüssel teilen Sie die erreichten Punkte durch die Maximalpunktzahl und lesen die Note an festen Prozentgrenzen ab. Ab 92 % gibt es eine 1, ab 81 % eine 2, ab 67 % eine 3, ab 50 % eine 4, ab 30 % eine 5, darunter eine 6. Der Notenrechner berechnet das automatisch — einfach Punkte eingeben und die Note ablesen.
Der Notenrechner bietet den IHK-Notenschlüssel (Dezimalnoten 1,0–6,0), Notenpunkte (15–0, für die Oberstufe an Gymnasien) mit optionaler Anzeige traditioneller Noten, sowie lineare Schlüssel mit 50 %, 45 % und 40 % Bestehensgrenze. Zusätzlich können Sie eigene Notenschlüssel mit individuellen Prozentgrenzen erstellen.

4. Datenverarbeitung in der App

Welche Daten werden verarbeitet? Innerhalb der App können vom Nutzer folgende Daten eingegeben werden:

  • Klassen- und Fachbezeichnungen, individuelle Notenschlüssel
  • Namen oder Pseudonyme von Schülerinnen und Schülern
  • Punktzahlen, Notenpunkte, mündliche Noten, Teilnoten
  • Klausur- und zeugnisbezogene Berechnungen
  • App-Einstellungen (z. B. Anzeige traditioneller Noten)

Lokale Speicherung. Sämtliche Daten werden in einer SQLite-Datenbank im Sandbox-Verzeichnis der App auf dem Endgerät gespeichert. iOS und iPadOS schotten dieses Verzeichnis pro App ab; ein Zugriff durch andere Apps ist betriebssystemseitig ausgeschlossen.

Keine Übermittlung an den Anbieter. Die App enthält keine Funktion, die Inhalte automatisch an den Anbieter überträgt. Es werden keine Cloud-Dienste, Analyse-Tools, Crash-Reporter, Tracking-Pixel oder Werbenetzwerke eingesetzt. Eine Internetverbindung ist für die Kernfunktion nicht erforderlich.

Backup-Funktion. Auf ausdrückliche Anweisung des Nutzers erstellt die App eine JSON-Datei mit allen lokalen Daten und übergibt sie an die System-Teilen-Funktion (iOS Share Sheet). Über welchen Kanal die Datei ausgeleitet wird — AirDrop, E-Mail, iCloud Drive, Messenger — entscheidet ausschließlich der Nutzer. Der Anbieter erhält dabei zu keinem Zeitpunkt Kenntnis von der Datei oder ihrem Inhalt.

In-App-Bewertungsdialog. Die App kann in seltenen Fällen das vom Betriebssystem bereitgestellte Bewertungs-Dialogfenster anzeigen (Apple StoreKit „requestReview" bzw. Google Play In-App Review). Dieser Dialog wird vom App Store gesteuert; der Anbieter erhält ausschließlich aggregierte Sternebewertungen und freiwillig hinterlassene Texte ohne Personenbezug.
